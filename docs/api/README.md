## 概述

本文档详细描述了数据中心管理系统（EAMS）的API接口规范，供后端开发团队参考实现。接口采用RESTful设计风格，使用JSON作为数据交换格式。

## 接口基本信息

- **基础URL**: `/api/v1`
- **认证方式**: Bearer Token
- **内容类型**: application/json
